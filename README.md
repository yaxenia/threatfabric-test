# Behavioral User Identification via Mobile Motion Patterns

The goal of the project is to develop a behavioral analytics system for identifying users based on unique mobile device movement patterns.

## Repository contents

- **Report (PDF):** [`Test_assessment_report.pdf`](./Test_assesment_report.pdf)  
  Project description, methodology, experiments, and results.

- **Final predictions / submission file:** [`submission.csv`](./submission.csv)  
  The final output file.

- **Code (notebook):** [`user_prediction.ipynb`](./user_prediction.ipynb)  
  Data preparation, feature engineering, model training, feature selection, and generation of the final submission.
